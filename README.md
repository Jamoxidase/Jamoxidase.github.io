I am using D3 to power an interactive visualization of real data from a drug discovery side-project I am working on. 


Point-clouds are really interesting data types for their ability to capture an object's 3-D features, and in this case to encode additional metadata. 


The point cloud data used in this demonstration was generated by [some code I wrote](https://github.com/Jamoxidase/MachineLearning/blob/main/drugPointCloud01.ipynb) as part of the data-preprocessing step of my deep learning approach. I utalized rdKit (a chemoinformatics library) to determine molecular geometry and NumPy to project points into space. Point charges in the molocule are represented by a scaling factor in point-cloud generation.
